# 添加新的团队成员

###### 1. 在 Adsforce 应用页面，选择团队成员；

![select-members](select-members.png)

###### 2. 点击“添加成员”；

![add-members](add-members.png)

###### 3. 在“添加成员”窗口，输入需添加的成员信息；

![fill-in-information](fill-in-information.png)

###### 4. 点击“保存”；

###### 5. 可在“团队成员”列表下，查看成员信息。
