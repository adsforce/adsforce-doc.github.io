Get Started

Get Started

1. Adsforce Access Process

1.1. View Demo

1. Visit Adsforce and click "View Demo".

2. Enter your information and click the "Contact Me" to request a Demo.

After clicking Demo and filling in required information, we will contact and send demo to you. Meanwhile, we will help you select the most cost-effective package based on your business size and budget.

[warning] Note

  • Currently only advertisers can integrate;
  • For media sources and agents partners interested in cooperation, please send email to contact@upltv.com.

1.2. Select A Mode and Use Adsforce

Select a mode:

  • Standard SaaS
  • Customized

1.2.1. Standard SaaS Mode

Adsforce server offers multiple features to advertisers, including attribution, reporting, ad revenue tracking, cohort analysis, anti-fraud etc. To begin your journey with Adsforce, please follow steps below:

  • Sign-up for a new Adsforce account or add team members to existing accounts;
  • Add your app to the Adsforce dashboard;
  • Download and integrate the SDK: Android, iOS or Unity;

  • Media Source Configurations (Facebook and Google, etc.);

  • Reporting.

1.2.2. Customized Mode

We provide customized mode solution for key accounts. Please email contact@upltv.com for more information.
