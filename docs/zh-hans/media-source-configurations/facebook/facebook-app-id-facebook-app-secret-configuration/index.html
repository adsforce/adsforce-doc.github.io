Facebook 应用编号和密钥配置

1. 概述

  • 在 Adsforce 中配置 Facebook 应用编号及密钥；
  • 参数由 Facebook 提供，是完成集成所必需的。

2. 配置

1. 打开 https://developers.facebook.com/apps，选择自己需要配置的应用。

­­

2. 点击“设置” > “基本”，查看并复制应用编号及密钥。
